Pixel Crawler is a very fun 2D top-down dungeon crawling game!
In this game you will search a randomly generated dungeon for loot, but watch out for the enemies!
There are two classes that you can choose from, Knight and Axeman. The Axeman if powerful, but the Knight is beefy.
There are several different variations of enemies, from skeletons to mimics!
Have fun playing the game!

**How to Play:**
Use **WASD** or **Arrow Keys** to move.
**Left Click** to attack, and **Right Click** for special.
On the knight the special is a shield, while the axeman has a swinging attack.
Use **X** to interact with any chests you see to get money.
Have fun exploring the randomly generated rooms!
The rest of the stuff should just be click-based.
